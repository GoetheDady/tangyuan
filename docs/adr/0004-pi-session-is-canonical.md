# Pi session 是会话唯一真相

会话消息和标题、创建时间、更新时间等元数据以 Pi Agent 原生 session 文件为唯一真相。汤圆的 `sessions/index.json` 只保存查询所需索引和 Pi session 无法表达的汤圆专属扩展数据；其中任何可由 Pi session 推导的字段都只是缓存，必须能够重建，不能反向覆盖 Pi session。

刷新索引时，以 Pi session 更新可推导字段，同时按会话标识保留仍有效会话的汤圆专属扩展数据。这样避免双写产生两份相互冲突的会话事实。

如果 Pi session 不再存在，该会话即视为已删除，对应索引条目和汤圆专属扩展数据一并清理。MVP 不保留脱离原生会话的孤立扩展数据，也不提供恢复站。
