# 汤圆主题色

## 决策

汤圆第一版先固定这套颜色，不继续推进 UI 设计稿。

主题方向：

- 糯米白做大面积背景。
- 米浆白做浮层和输入区。
- 芝麻黑做品牌主色和关键操作。
- 红豆色只用于危险、错误或轻提示。
- 豆青色只用于成功、连接、运行中状态。

**主题色** 指产品长期使用的一组基础颜色。

**语义色** 指按用途命名的颜色，例如 `background` 表示背景色，`danger` 表示危险状态色，而不是只按视觉名称命名。

## 色板

| 语义名 | 中文名 | 色值 | 用途 |
| --- | --- | --- | --- |
| `background` | 糯米白 | `#FBF7EF` | 页面主背景 |
| `surface` | 米浆白 | `#FFFDF8` | 卡片、面板、输入区 |
| `surface-soft` | 浅米灰 | `#F1E9DE` | 选中态、次级浮层 |
| `text` | 芝麻深黑 | `#211B16` | 主文字 |
| `text-muted` | 温灰褐 | `#7D7166` | 次级文字、说明文字 |
| `border` | 米灰线 | `#DED2C2` | 边框、分割线 |
| `brand` | 黑芝麻 | `#1F1A15` | 品牌主色、主按钮 |
| `brand-soft` | 芝麻糊 | `#3A322A` | 品牌悬停态、强调文字 |
| `success` | 豆青色 | `#5F8B70` | 运行中、连接成功、完成 |
| `danger` | 红豆色 | `#B66C63` | 错误、取消、危险操作 |
| `focus` | 糯米深边 | `#8B7563` | 输入框焦点、键盘焦点 |

## CSS 变量

```css
:root {
  --color-background: #fbf7ef;
  --color-surface: #fffdf8;
  --color-surface-soft: #f1e9de;
  --color-text: #211b16;
  --color-text-muted: #7d7166;
  --color-border: #ded2c2;
  --color-brand: #1f1a15;
  --color-brand-soft: #3a322a;
  --color-success: #5f8b70;
  --color-danger: #b66c63;
  --color-focus: #8b7563;
}
```

## 使用约束

- 不使用桂花金。
- 不使用紫色渐变。
- 不做大面积黑色界面。
- 不把红豆色当主按钮色。
- 不把豆青色当装饰色，它只表示正向状态。
- 黑芝麻颗粒感以后只能作为局部品牌细节，不进入当前颜色决策。
