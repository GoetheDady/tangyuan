# Triage 标签

工程技能内部使用五个标准 triage 角色。本文件把这些角色映射到本仓库实际使用的 GitHub 中文标签。

| mattpocock/skills 角色 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `待评估` | 维护者需要评估这个 issue |
| `needs-info` | `需要更多信息` | 等待报告者补充信息 |
| `ready-for-agent` | `可交给 Agent` | 需求已经足够清楚，Agent 可以接手 |
| `ready-for-human` | `需要人工处理` | 需要人工实现、判断或确认 |
| `wontfix` | `不处理` | 不会处理 |

当技能提到某个标准角色时，使用本表右侧对应的中文标签。
