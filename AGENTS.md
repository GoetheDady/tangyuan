# 汤圆 Agent 说明

## Agent skills

### Issue tracker

Issues 和 PRD 使用 GitHub Issues，仓库是 `git@github.com:GoetheDady/tangyuan.git`；外部 PR 不作为 triage 请求入口。详见 `docs/agents/issue-tracker.md`。

### Triage labels

Triage 角色映射为中文 GitHub 标签：`待评估`、`需要更多信息`、`可交给 Agent`、`需要人工处理`、`不处理`。详见 `docs/agents/triage-labels.md`。

### Domain docs

本仓库使用 single-context 领域文档结构：根目录 `CONTEXT.md` 加 `docs/adr/`。详见 `docs/agents/domain.md`。
